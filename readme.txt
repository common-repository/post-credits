Adds a footer to the post, stating the copyright notice, the year, and the author. 

== Description ==

This plugin adds a line at the end of each post, stating:

* Copyright (or Creative Commons licence) notice
* Year
* Author

It also permits to add a suplementary text line where you can type whatever you want.

The plugin is totally customizable via "Options" section in you WordPress Panell. There you can change:

* Text Alignment (Left/Right/Center)
* Font Stile (Normal/Italic/Bold)
* Font Effects (Normal/Small Caps)
* Text Size (from 10 to 30 points)
* Text Color (Black/Grey/Dark Red/Orange/Green/Yellow)
* Include year or not
* Include copyright symbol (&copy;) or Creative Commons (CC)
* View the credits in Posts, in Pages, in Posts and Pages, or not view it.


== Installation ==


1. Upload `post-credits` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress. No further actions needed.

3. To configure it, go to the "Post Credits" link in the "Options" menu of your WordPress panel.
